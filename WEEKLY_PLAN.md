## 🔧 开发环境要求

### API服务
```bash
cd services/api
npm run start:dev
```
- 端口: 8000
- 数据库: PostgreSQL (需配置)
- Redis: 可选 (本阶段非必需)

### 前端服务
```bash
cd apps/web
npm run dev
```
- 端口: 3000
- API地址: http://localhost:8000

### 数据库
```bash
# 连接数据库
psql -h localhost -U postgres -d photo_platform

# 创建测试数据
\i infrastructure/database/schema.sql
```
