## 🐛 已知Bug与修复记录

### 已修复（2026-03-15）
- ✅ 数据库未连接时 API 返回 500 — 已增加全局异常过滤器，返回 503 友好提示
- ✅ 大文件上传超时 — 已设置 body 50MB、请求超时 120s
- ✅ Token 过期后无法操作 — 前端 apiFetch 在 401 时清除登录态并跳转登录页

- ✅ 列表/详情空值导致异常 — 接口返回做空值防护（items ?? [] 等）
- ✅ 图片加载失败无占位 — SafeImage 组件 onError 显示占位
