## 运行说明

```bash
# 前端
cd apps/web && npm run dev

# 需认证/个人中心/文章时启动 API
cd services/api && npm run start:dev

# 运行 E2E（端口非 3000 时设置 PLAYWRIGHT_BASE_URL）
cd apps/web && npm run test:e2e
```

## 备注

- 个人中心编辑用例依赖 API 返回登录 user，否则编辑页会重定向到登录，用例会跳过保存断言。
- 本地未配置自动启动 webServer，需先手动 `npm run dev` 再跑 E2E。
